# chatIO
A template chat application made using Node.js, Express, HTML5/CSS3, Socket.IO, and JQuery. Ready for Bootstrap customisation and deployment.

To get this up and running:
1) cd chatIO
2) npm install
3) npm start (app runs on localhost:3000)
